Introduction

This is the HandyCAN interlink node firmware for STM32F10x
A interlink node serves as a bridge between a PC and the HandyCAN network
There is no limit to the amount of interlink nodes in a setup
They are not required.
If the CAN bus is run at over 500kbaud, the STM32F10 is not powerfull enough to decode all packages.
When this is required, use the STM32F4 interlink node

For simplicity and because the interlink node firmware is not very complex,
it is entirely implemented in main.c.

Usage

One or more interlink nodes are connected to the CAN bus and, via a fast UART, to the PC.
