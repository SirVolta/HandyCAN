1 # PC Application
2 The interlink node sends encapsulated CAN packets over the UART to the PC,
3 and transmits encapsulated CAN packets to the interlink node, which in turn sends them over the CAN bus.
4 This way the PC software has maximum flexibility, and the relatively processor intensive job of
5 decoding all CAN packets is done by the PC, not the interlink node.
6 
7 This folder contains a python implementation of the [interlink protocol described here.](https://github.com/SirVolta/HandyCAN/tree/master/doc/protocol)
8 
9 Example usage of the python module will be in the examples folder.
